# PDF族谱导入功能 - 快速开始指南

## 📖 功能概述

PDF族谱导入功能允许您从PDF格式的族谱文档中自动提取人员信息并导入到系统中。

**支持功能**:
- ✅ 上传PDF文件（最大50MB）
- ✅ 自动解析PDF文本内容
- ✅ 智能提取人员信息（姓名、性别、辈分、日期等）
- ✅ 在线预览和校对
- ✅ 一键导入数据库

## 🚀 快速开始

### 1. 启动服务

**启动后端**:
```bash
cd e:\GeneaSphere\apps\server
pnpm run dev
```

**启动前端**:
```bash
cd e:\GeneaSphere\apps\web
pnpm run dev
```

### 2. 访问导入页面

打开浏览器访问:
```
http://localhost:5173/pdf-import
```

### 3. 导入流程

#### 步骤1: 上传PDF
- 拖拽PDF文件到上传区域，或点击选择文件
- 支持格式: `.pdf`
- 最大大小: `50MB`
- 点击"开始解析"按钮

#### 步骤2: 等待解析
- 系统自动解析PDF并提取人员信息
- 进度条显示解析进度
- 通常几秒到几十秒（取决于文件大小）

#### 步骤3: 预览校对
- 查看提取到的人员记录列表
- 关注置信度评分:
  - 🟢 高置信度(≥90%): 数据可靠
  - 🟡 中置信度(70-89%): 建议检查
  - 🔴 低置信度(<70%): 需要重点校对
- 点击表格单元格编辑数据:
  - 修改姓名
  - 选择性别
  - 调整辈分
  - 选择日期

#### 步骤4: 确认导入
- 点击"确认导入"按钮
- 系统批量导入数据到数据库
- 显示导入结果（成功数、失败数）
- 查看族谱树验证导入结果

## 📋 支持的PDF类型

### ✅ 完全支持
- 文本PDF（可直接复制文字的PDF）
- 电子版族谱文档
- Word导出的PDF

### ⚠️ 部分支持（需要OCR，开发中）
- 扫描件PDF（纸质族谱扫描）
- 图片PDF
- 手写体PDF

### ❌ 暂不支持
- 加密PDF（需要先解密）
- 纯图片PDF（无文字层）

## 💡 使用技巧

### 提高识别准确率
1. **选择文本PDF**: 优先使用电子版而非扫描件
2. **清晰格式**: 确保PDF中人员信息格式规整
3. **分段导入**: 大文件可以拆分成多个小文件
4. **仔细校对**: 重点关注低置信度记录

### 常见问题

**Q: 解析后没有找到人员信息？**
A: 可能是扫描件PDF，需要等待OCR功能集成。或者PDF格式不符合常见族谱格式。

**Q: 导入速度慢？**
A: 大文件解析需要时间，请耐心等待。建议文件大小控制在20MB以内。

**Q: 识别准确率低？**
A: 当前版本使用规则引擎，对标准格式识别率高。复杂格式需要手动校对。

**Q: 导入后可以修改吗？**
A: 可以在族谱树页面修改已导入的人员信息。

## 🔧 API接口

如果您是开发者，可以直接调用API:

### 上传PDF
```http
POST /api/import/pdf/upload
Content-Type: multipart/form-data

file: [PDF文件]
clan_id: 123
user_id: "uuid-here"
```

### 查询状态
```http
GET /api/import/pdf/task/{taskId}/status
```

### 获取预览
```http
GET /api/import/pdf/task/{taskId}/preview
```

### 执行导入
```http
POST /api/import/pdf/task/{taskId}/execute
{
  "user_id": "uuid-here",
  "clan_id": "123"
}
```

