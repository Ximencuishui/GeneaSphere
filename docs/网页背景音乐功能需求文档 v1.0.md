# 网页背景音乐功能需求文档 v1.0

**文档版本**：v1.0  
**对应系统版本**：GeneaSphere Platform v1.0  
**前置依赖**：前端路由系统、音频播放能力  
**面向对象**：开发团队、测试团队  

---

## 1. 功能概述

### 1.1 背景

GeneaSphere 作为家族文化传承平台，承载着浓厚的怀旧与情感氛围。在夜间浏览家谱、查看老照片、回顾家族历史时，添加自然环境音效（如虫鸣、蛙声）可以：

- 增强沉浸式体验，让用户仿佛回到故乡的夜晚
- 营造温馨、宁静的氛围，特别适合夜间浏览时光长廊
- 为家族记忆增添感官维度，强化情感连接

### 1.2 目标

- 提供可选的背景音乐播放功能，默认静音，用户可手动开启
- 支持夜间主题音效（虫鸣、蛙声等自然环境音）
- 用户可控制音量、暂停/播放、切换音效
- 音效播放不影响页面其他功能的使用

### 1.3 用户故事

- **作为夜间浏览者**：晚上查看家族老照片时，开启虫鸣蛙声音效，仿佛回到了童年的乡村夜晚
- **作为长辈**：在安静的夜晚回忆往事时，背景音乐让氛围更加温馨
- **作为年轻用户**：通过音效体验祖辈生活的环境，增强对家族历史的共鸣

## 2. 功能列表

### 2.1 音效控制面板

#### 2.1.1 入口位置

- 在页面右下角固定显示"背景音乐"控制按钮（图标：🎵）
- 按钮状态：
  - 未播放：灰色图标 + "添加背景音乐"
  - 播放中：绿色图标 + 音量图标

#### 2.1.2 控制面板内容

点击按钮后展开迷你控制面板，包含：

| 控件 | 类型 | 说明 |
|------|------|------|
| 播放/暂停 | 按钮 | 控制音效播放状态 |
| 音量调节 | 滑块 | 0-100% 音量控制 |
| 音效选择 | 下拉菜单 | 可选：夜间虫鸣、田野蛙声、夏日夜晚（混合） |
| 定时关闭 | 下拉菜单 | 可选：15分钟、30分钟、1小时、2小时、不关闭 |

### 2.2 音效资源

#### 2.2.1 预设音效列表

| 音效名称 | 描述 | 适用场景 |
|----------|------|----------|
| 夜间虫鸣 | 远处蟋蟀、纺织娘的叫声 | 深夜浏览、回忆往事 |
| 田野蛙声 | 稻田青蛙的叫声 | 乡村记忆、夏日怀旧 |
| 夏日夜晚 | 虫鸣+蛙声+微风混合 | 综合场景、沉浸体验 |

#### 2.2.2 音频技术要求

- 格式：MP3 或 OGG（浏览器兼容）
- 时长：至少 3-5 分钟循环（无缝循环）
- 音质：128kbps 以上
- 文件大小：单个文件 < 2MB
- 循环：支持无缝循环播放（loop）

## 3. 界面交互设计

### 3.1 控制面板 UI 布局

```
┌─────────────────────────────┐
│  🎵 背景音乐                 │
├─────────────────────────────┤
│  [▶ 播放] [⏸ 暂停]           │
│                             │
│  音量：[━━━━━━●━━━━━━] 60%  │
│                             │
│  音效：[夜间虫鸣 ▼]          │
│                             │
│  定时：[30分钟 ▼]            │
│                             │
│  [✕ 关闭面板]                │
└─────────────────────────────┘
```

### 3.2 交互行为

1. **首次访问**：
   - 默认静音状态
   - 显示提示："点击添加背景音乐，体验沉浸式家族回忆"

2. **点击播放**：
   - 按钮图标变为播放状态
   - 控制面板自动展开
   - 开始播放默认音效（夜间虫鸣）

3. **切换音效**：
   - 当前播放自动停止
   - 新音效立即开始播放
   - 保持当前音量设置

4. **定时关闭**：
   - 到达设定时间后自动停止播放
   - 按钮恢复为未播放状态
   - 显示提示："背景音乐已自动关闭"

## 4. 技术实现方案

### 4.1 前端架构

#### 4.1.1 组件结构

```
apps/web/src/
├── components/
│   └── BackgroundMusic/
│       ├── MusicControl.vue          # 主控制面板组件
│       ├── MusicButton.vue           # 浮动按钮组件
│       └── VolumeSlider.vue          # 音量控制组件
├── composables/
│   └── useBackgroundMusic.ts         # 音乐播放逻辑（Composition API）
├── assets/
│   └── audio/
│       ├── night-crickets.mp3        # 夜间虫鸣
│       ├── field-frogs.mp3           # 田野蛙声
│       └── summer-night.mp3          # 夏日夜晚
└── stores/
    └── music.ts                      # Pinia 状态管理
```

#### 4.1.2 状态管理（Pinia Store）

```typescript
interface MusicState {
  isPlaying: boolean;          // 是否正在播放
  volume: number;              // 音量 0-100
  currentTrack: string;        // 当前播放的音效名称
  timer: number | null;        // 定时关闭剩余时间（分钟）
  isEnabled: boolean;          // 用户是否启用背景音乐
}
```

### 4.2 音频播放实现

#### 4.2.1 使用 HTML5 Audio API

```typescript
// 伪代码示例
const audio = new Audio();
audio.loop = true;  // 无缝循环
audio.volume = state.volume / 100;
audio.src = `/audio/${state.currentTrack}.mp3`;

// 播放控制
audio.play();   // 播放
audio.pause();  // 暂停
```

#### 4.2.2 浏览器兼容性处理

- 自动播放限制：需要用户交互后才能播放（符合现代浏览器策略）
- 移动端优化：iOS/Android 需要特殊处理音频上下文
- 降级方案：不支持 Audio API 的浏览器显示提示信息

### 4.3 性能优化

- 音频文件预加载：用户首次点击后加载音频资源
- 懒加载：非当前播放的音效不占用内存
- 循环优化：使用无缝循环避免播放间隔
- 资源缓存：浏览器缓存已加载的音频文件

## 5. 数据架构

### 5.1 本地存储（LocalStorage）

保存用户偏好设置：

```json
{
  "music_enabled": true,
  "music_volume": 60,
  "music_track": "night-crickets",
  "music_timer": 30
}
```

### 5.2 不需要后端支持

- 所有音频资源存储在前端静态资源目录
- 用户偏好存储在本地浏览器
- 不涉及数据库表修改

## 6. 错误处理与边界条件

| 场景 | 处理方式 |
|------|----------|
| 音频文件加载失败 | 显示提示"音效加载失败，请稍后重试"，按钮恢复为未播放状态 |
| 浏览器不支持自动播放 | 显示提示"请点击按钮手动播放背景音乐" |
| 移动端切换后台 | 暂停播放，返回前台后询问是否继续 |
| 网络断开 | 已缓存的音频可继续播放，未缓存的显示加载失败 |
| 定时关闭触发 | 平滑淡出（2秒内音量渐降至0），然后停止播放 |

## 7. 非功能性需求

- **性能**：音频加载时间 < 2s（128kbps，< 2MB）
- **兼容性**：支持 Chrome、Firefox、Safari、Edge 最新版
- **移动端**：iOS Safari、Android Chrome 支持
- **无障碍**：支持键盘操作（Tab 切换、Enter 播放/暂停）
- **用户体验**：所有操作响应 < 200ms

## 附录：音频资源获取建议

### 合法音源推荐

1. **Freesound.org**（CC0 协议，免费商用）
   - 搜索关键词：crickets night、frog chorus、summer evening
   
2. **Freesounds Library**（公共领域）
   - 确保下载时标注为 Public Domain 或 CC0

3. **自行录制**
   - 使用录音设备在乡村夜间录制
   - 需要降噪处理和循环编辑

### 音频处理工具

- **Audacity**（免费）：剪辑、降噪、循环处理
- **FFmpeg**：格式转换、比特率调整
- **Online Audio Converter**：快速格式转换

### 循环处理技巧

1. 选择 3-5 分钟自然片段
2. 在 Audacity 中使用 "Crossfade Loop" 插件
3. 确保首尾过渡自然，无明显断点
4. 导出为 MP3（128kbps）或 OGG 格式
