根脉云谱（GeneaSphere）用户中心页面需求文档 v1.0

文档版本：v1.0  

对应系统版本：GeneaSphere Platform v1.0  

前置依赖：用户系统、家族系统、时光长廊、工具箱、印刷订单、小组讨论等模块  

面向对象：开发团队、测试团队、UI设计师  

1. 功能概述

1.1 背景

用户中心是每个家族成员管理个人数据、查看活动记录、配置隐私偏好、访问增值服务的统一入口。它将分散在平台各处的个人相关信息聚合到一起，提供清晰的导航和操作入口。

1.2 目标

• 展示用户基本信息（头像、姓名、所属家族、角色）。

• 汇总用户的各类数据（照片数、标注数、音像墙数、订单数、小组数）。

• 提供快捷入口：我的家族树、我的时光长廊、我的工具箱、我的订单、我的小组、我的标注、我的音像墙。

• 支持个人资料编辑、密码修改、隐私设置、通知偏好。

• 展示最近动态（如最近上传的照片、最近的标注、最近的寻亲匹配）。

2. 角色与权限

角色 权限

普通成员 可查看和编辑自己的信息，管理自己的内容

家族管理员 在用户中心基础上，额外显示“家族管理后台”入口

游客 不可访问用户中心（需登录）

3.2 左侧面板（固定区域）

元素 说明

用户头像 圆形，可点击更换头像

用户姓名 显示真实姓名或昵称

所属家族 显示家族名称，点击可跳转到家族首页

角色标签 如“管理员”、“编辑者”、“成员”

菜单列表 垂直导航，点击切换右侧内容

3.3 右侧内容区（按菜单切换）

3.3.1 个人资料

• 展示字段：头像、姓名、性别、出生日期、手机号（脱敏）、邮箱（可选）。

• 可编辑字段：头像、姓名、邮箱。

• 操作：修改密码（跳转至修改密码页）。

3.3.2 我的家族

• 展示用户所属的所有家族（支持多家族？目前设计一个用户属于一个家族，但可预留）。

• 每个家族卡片显示：家族名称、成员数、角色、最后活跃时间。

• 点击进入该家族的族谱树或家族空间。

3.3.3 我的时光

• 展示用户上传的所有照片/视频（分页）。

• 支持按年份、按家族筛选。

• 点击进入照片详情页。

3.3.4 我的工具箱

• 展示用户使用AI工具箱的历史记录（最近20条）。

• 每条记录显示：工具类型、处理时间、处理结果缩略图。

• 点击可查看原图和结果图对比。

3.3.5 我的订单

• 展示用户的印刷订单列表（分页）。

• 字段：订单号、下单时间、金额、状态（待支付/印刷中/已发货/已完成/已退款）。

• 点击查看订单详情（含物流信息）。

3.3.6 我的小组

• 展示用户加入的所有讨论组。

• 每个小组卡片显示：名称、成员数、最后活跃时间、未读话题数。

• 点击进入小组详情页。

3.3.7 我的标注

• 展示用户在时光长廊中做出的所有照片标注（分页）。

• 每条标注显示：照片缩略图、地点、时间段、关系状态、备注。

• 点击跳转到对应的照片详情页。

3.3.8 我的音像墙

• 展示用户生成的“历史音像墙”视频列表。

• 每个视频卡片显示：目标人物、生成时间、时长、状态（生成中/已完成/失败）。

• 点击播放/下载。

3.3.9 设置

• 隐私设置：

  • 是否允许跨家族寻找小伙伴（开关）。

  • 是否公开童年地点（开关）。

  • 是否允许他人通过照片找到我（开关）。

  • 是否允许我的标注用于匹配（开关）。

• 通知偏好：

  • 接收站内信通知（开关）。

  • 接收短信通知（开关，需家族管理员有权限）。

• 账号安全：

  • 修改密码。

  • 绑定手机（如未绑定）。

  • 注销账号（需二次确认，并通知家族管理员）。

4. 数据接口

接口 方法 说明

/api/user/profile GET 获取当前用户个人信息

/api/user/profile PUT 更新个人信息

/api/user/photos GET 获取用户上传的照片列表（分页）

/api/user/tool-history GET 获取AI工具箱使用记录

/api/user/orders GET 获取订单列表

/api/user/groups GET 获取加入的小组列表

/api/user/annotations GET 获取照片标注列表

/api/user/videos GET 获取音像墙视频列表

/api/user/settings GET/PUT 获取/更新用户设置

5. 交互细节

• 头像更换：点击头像弹出上传裁剪组件，支持jpg/png，最大5MB，自动生成圆形裁剪。

• 修改密码：需要输入旧密码、新密码、确认新密码，符合强度要求（8位以上，含字母和数字）。

• 注销账号：弹出确认对话框，提示“注销后您的所有数据将被删除，此操作不可撤销”，输入“确认注销”文字后方可提交。

• 通知铃铛：顶部导航栏的通知铃铛显示未读站内信数量，点击展开下拉列表。

6. 错误处理与边界条件

场景 处理方式

用户未登录访问用户中心 跳转到登录页

用户无任何数据（如未上传照片） 显示“暂无数据”占位图，并提供“去上传”按钮

修改密码时旧密码错误 提示“旧密码不正确”

注销账号时用户是家族唯一管理员 提示“请先转让管理员权限再注销”

7. 非功能性需求

• 性能：页面首次加载 < 2s，列表分页加载 < 1s。

• 兼容性：PC端和移动端自适应（移动端左侧菜单折叠为汉堡菜单）。

• 安全性：所有接口需校验用户Token；敏感操作（修改密码、注销）需二次确认。

8. 验收标准（DoD）

1. 用户登录后点击右上角头像进入用户中心，左侧显示用户信息和菜单。
2. 点击“个人资料”可编辑姓名和邮箱，修改后保存成功。
3. 点击“我的时光”显示用户上传的照片列表，支持分页。
4. 点击“我的订单”显示订单列表，状态正确。
5. 在“设置”中关闭“跨家族寻找小伙伴”后，寻找小伙伴功能不再返回跨家族结果。
6. 修改密码功能正常工作，旧密码错误时有提示。

9. 与现有模块的关系

模块 关系

用户系统 提供基础用户数据和认证

时光长廊 “我的时光”和“我的标注”依赖此模块

AI工具箱 “我的工具箱”依赖此模块

印刷订单 “我的订单”依赖此模块

小组讨论 “我的小组”依赖此模块

音像墙 “我的音像墙”依赖此模块

寻找小伙伴 “设置”中的隐私开关影响此模块
