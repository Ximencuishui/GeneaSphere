用户中心 · 直系血缘视频生成功能需求 v1.0

本文档是对《用户中心页面需求文档 v1.0》和《历史音像墙功能需求文档 v1.0》的补充，新增“直系血缘视频生成”功能，专注于解决“我从哪里来”的核心问题。

1. 功能概述

1.1 背景

现有“历史音像墙”功能是为单个成员生成包含其直系血脉（上下多代）的视频，但用户希望更灵活地控制范围：只生成自己的直系祖先和后代，不包含旁支，并且可以自由设定“从第几代祖先到第几代后代”。这个功能更精准地回答“我从哪里来”——用户只需选择自己作为中心，系统沿父系/母系向上追溯指定代数，向下延展指定代数，仅展示这条直线上的图文音像。

1.2 目标

• 用户以自己为中心，指定向上追溯的代数（如到始祖或第10代）和向下延展的代数（如到孙子辈）。

• 系统仅提取直系血缘（父系+母系，或仅父系，用户可选）的相关图文音像素材。

• 生成一段视频（类似历史音像墙），展示这条血脉线上的关键人物、照片、地点、时间线。

• 支持预览、下载、分享，并可申请纳入族谱印刷版。

1.3 用户故事

• 作为中年用户：我想知道“我从哪里来”，生成一个视频，从爷爷的爷爷一直到我的孙子，只显示直系，不显示叔叔伯伯家。

• 作为年轻人：我想看看爸爸这边的祖先和妈妈这边的祖先分别是什么样的，可以分别生成两条线。

2. 功能详情

2.1 触发入口

• 用户中心 → 工具箱：新增“直系血缘视频”卡片。

• 个人详情页：在“历史音像墙”按钮旁增加“直系血缘视频”按钮。

• 族谱树：右键点击人物节点，菜单中增加“生成直系血缘视频”。

2.2 参数设置

参数 说明 默认值

中心人物 默认为当前登录用户，可手动选择其他家族成员 当前用户

追溯方向 父系（仅父亲血脉）/ 母系（仅母亲血脉）/ 双系（父母双方合并） 父系

向上代数 从中心人物向上追溯的代数（1-20） 5

向下代数 从中心人物向下延展的代数（0-10，0表示不包含后代） 3

包含配偶 是否包含直系祖先的配偶（如曾祖母） 是

视频风格 同历史音像墙（怀旧/现代/黑白） 怀旧

2.3 素材收集逻辑

• 系统根据中心人物和设置的方向，沿父系/母系向上查找指定代数的直系祖先（不包括兄弟姊妹及其后代）。

• 向下查找中心人物的直系后代（子女、孙子女等，不包括侄甥等旁系）。

• 从 media_archive 和 user_photos 中提取这些人物的照片/视频，按时间排序。

• 若无照片，使用人物头像占位，显示姓名和生卒年。

2.4 视频生成

• 复用历史音像墙的 AI 视频生成服务。

• 视频标题自动生成，如“张三的直系血脉：从高祖到孙子”。

• 视频内容按时间线从最古老的祖先开始，逐渐推进到中心人物，再延伸到后代。

• 每个节点显示人物姓名、生卒年、与中心人物的关系（如“高祖：张大山 1800-1875”）。

2.5 排队与付费

• 与历史音像墙共用排队队列和付费优先机制。

• 免费用户每月可生成 2 条，付费用户不限。

3. 数据架构

3.1 新增表

-- 直系血缘视频项目表
CREATE TABLE lineage_video_projects (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  user_id BIGINT NOT NULL,
  center_person_id BIGINT NOT NULL,
  direction ENUM('paternal', 'maternal', 'both') DEFAULT 'paternal',
  up_generations INT DEFAULT 5,
  down_generations INT DEFAULT 3,
  include_spouse BOOLEAN DEFAULT TRUE,
  style VARCHAR(50) DEFAULT 'nostalgic',
  status ENUM('queued', 'processing', 'completed', 'failed') DEFAULT 'queued',
  video_url VARCHAR(500) NULL,
  duration_seconds INT NULL,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  completed_at TIMESTAMP NULL
);


3.2 与现有表的关系

• 复用 video_projects 表（历史音像墙）？建议新建独立表，因为逻辑不同（直系血缘 vs 个人全血脉）。但视频生成服务可复用同一套 AI 管道。

• 显示“血脉图谱”：以中心人物为圆心，向上/向下辐射的树状图（静态截图）。

5. 验收标准

1. 用户选择自己为中心，向上5代、向下3代，生成的视频包含高祖到孙辈的直系人物，无旁支。
2. 选择“父系”时，只显示父亲血脉；选择“母系”时只显示母亲血脉。
3. 视频按时间顺序播放，每个节点显示人物信息和照片。
4. 免费用户每月可生成2条，超出后提示购买次数包。
