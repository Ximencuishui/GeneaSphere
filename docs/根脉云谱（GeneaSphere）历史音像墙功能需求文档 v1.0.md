根脉云谱（GeneaSphere）历史音像墙功能需求文档 v1.0

文档版本：v1.0  

对应系统版本：GeneaSphere Platform v1.0  

前置依赖：数据架构 v1.0（persons、media_archive、media_person_links）、时光长廊模块、AI 服务  

面向对象：开发团队、测试团队、家族管理员  

1. 功能概述

1.1 背景

家族成员希望以更生动、感性的方式回顾先辈的一生——不再仅仅是静态的照片和文字，而是能将散落在时光长廊中的影像串联起来，形成一部“个人生命纪录片”。同时，许多家族希望在祭祖、聚会等场合播放这样的影片，增强家族凝聚力。

1.2 目标

• 允许用户选择一位家族成员（在世或已故），系统自动提取其直系血脉（向上至始祖，向下至最新一代）的所有关联影像和文字说明。

• 利用 AI 技术将这些素材自动剪辑、配音、配乐，生成一段 1-5 分钟的短视频（“历史音像墙”）。

• 考虑到服务器资源有限，生成任务采用排队机制，并提供付费优先通道（付费用户可跳过排队直接生成）。

• 生成的视频可下载、分享、收藏，并可申请纳入族谱印刷版的光盘/二维码附件。

2. 角色与权限

角色 权限

普通成员 可为本人或本家族任意成员发起生成请求（需消耗积分或排队）

付费用户 享有优先生成权（跳过排队），不限次数

家族管理员 可为本家族任何成员发起生成，可管理本家族的生成队列

游客 不可使用此功能

3. 功能列表

3.1 触发生成

3.1.1 入口

• 个人详情页：在“影像馆”区块增加“生成历史音像墙”按钮。

• 时光长廊：在家族时光长廊顶部增加“为我家族成员生成”入口。

• 族谱树：右键点击人物节点，菜单中增加“生成音像墙”。

3.1.2 选择目标人物

• 用户可通过搜索或从族谱树中选择一位成员。

• 系统自动识别其直系血脉范围：

  • 向上：所有直系祖先（父、祖父、曾祖父……直至始祖）。

  • 向下：所有直系后代（子女、孙子女……直至最新一代）。

  • 旁系：不包含（仅直系血脉，确保视频主题聚焦）。

3.1.3 素材收集

系统自动执行以下操作：
1. 从 media_archive 表中查询所有关联到目标人物及其直系血脉的照片/视频（通过 media_person_links）。
2. 按时间顺序（taken_year）排序。
3. 提取每张照片的描述文字（description 字段）和人物生平简介（bio_text）。
4. 若无照片，则使用人物头像占位，并显示姓名和生卒年。

3.1.4 视频生成

• 调用 AI 视频生成服务（如 Runway Gen-3、Pika、或自研模型）。

• 输入：图片序列 + 文字脚本（由 AI 根据照片描述和生平自动生成解说词）。

• 输出：MP4 格式，分辨率 1080p，时长根据素材数量自动调整（最短30秒，最长5分钟）。

• 风格：默认温馨怀旧风格（泛黄滤镜、柔和转场），可提供 2-3 种备选风格（如黑白复古、现代明亮）。

3.2 排队与付费优先

3.2.1 排队机制

• 所有生成请求进入全局队列（FIFO）。

• 队列状态显示在用户界面：当前排队人数、预计等待时间。

• 用户可取消自己的排队请求。

3.2.2 付费优先

• 付费用户（购买“优先生成券”或订阅 VIP）的请求直接插入队列头部，立即处理。

• 价格策略：

  • 单次优先生成：¥9.9

  • 月度 VIP（无限优先）：¥29.9

  • 年度 VIP（无限优先+高清无水印）：¥199

• 支付成功后，请求立即进入处理状态。

3.2.3 生成进度

• 生成过程中显示进度条（0% - 100%）。

• 支持后台生成，用户可离开页面，生成完成后通过站内信通知。

3.3 视频管理与分享

3.3.1 我的音像墙

• 用户个人中心新增“我的音像墙”页面，展示所有已生成的视频。

• 列表显示：目标人物姓名、生成时间、时长、状态（生成中/已完成/失败）。

• 支持播放、下载（MP4）、删除。

3.3.2 分享

• 生成分享链接（需登录且属于同一家族成员才能观看）。

• 支持下载到本地，用于线下播放。

• 可申请将视频二维码印制在族谱中（增值服务，对接印刷模块）。

4. 数据架构

4.1 新增表

-- 视频生成项目表
CREATE TABLE video_projects (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  clan_id BIGINT NOT NULL,
  requester_id BIGINT NOT NULL,           -- 发起人用户ID
  target_person_id BIGINT NOT NULL,       -- 目标人物ID
  status ENUM('queued', 'processing', 'completed', 'failed') DEFAULT 'queued',
  queue_position INT DEFAULT 0,           -- 排队位置（实时计算）
  priority BOOLEAN DEFAULT FALSE,         -- 是否为付费优先
  paid_amount DECIMAL(10,2) DEFAULT 0.00, -- 支付金额
  video_url VARCHAR(500) NULL,            -- 生成后的视频地址
  duration_seconds INT NULL,              -- 视频时长
  style VARCHAR(50) DEFAULT 'nostalgic',  -- 风格
  error_message TEXT NULL,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  completed_at TIMESTAMP NULL
);

-- 视频素材关联表（记录该视频使用了哪些照片/视频）
CREATE TABLE video_material_links (
  video_project_id BIGINT NOT NULL,
  media_id BIGINT NOT NULL,
  sequence_order INT NOT NULL,            -- 在视频中的顺序
  PRIMARY KEY (video_project_id, media_id)
);

-- 付费优先券/订阅表（简略）
CREATE TABLE user_vip_orders (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  user_id BIGINT NOT NULL,
  order_type ENUM('single', 'monthly', 'yearly'),
  amount DECIMAL(10,2),
  expires_at TIMESTAMP NULL,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);


4.2 索引建议

• video_projects：索引 (requester_id, status) 用于用户查询自己的项目。

• video_projects：索引 (status, priority, created_at) 用于队列排序。

5. 界面交互设计

5.1 生成入口（以个人详情页为例）

• 在“影像馆”区块下方增加“生成历史音像墙”卡片。

• 卡片显示当前人物姓名、直系血脉人数、可用照片数量。

• 点击后弹出对话框：

  • 选择视频风格（下拉选择）。

  • 显示预计排队时间（若非付费用户）。

  • 按钮：“开始生成（排队）” / “付费优先生成（¥9.9）”。

5.2 排队状态页

• 显示当前排队位置、预计等待时间。

• 进度条（如果已经开始处理）。

• 取消按钮。

5.3 视频播放页

• 全屏播放器，支持倍速、下载、分享。

• 下方显示视频信息：目标人物、生成时间、时长、使用素材数量。

• 评论区（可选，复用小组讨论功能）。

5.4 我的音像墙列表

• 卡片网格，每个卡片显示封面（视频第一帧）、人物姓名、时长、状态标签。

• 点击进入播放页。

6. 错误处理与边界条件

场景 处理方式

目标人物无任何关联照片 提示“暂无可用素材，请先上传照片”，并引导至上传页面

直系血脉人数过多（>100人） 限制最多选取前5代祖先和后3代后代，提示可手动调整范围

AI 生成失败（如模型超时） 自动重试一次，仍失败则标记为失败，退还付费金额（如有）

排队人数过多（>1000） 提示“当前繁忙，预计等待时间较长”，建议使用付费优先

视频生成中途取消 已消耗的计算资源不退费（付费用户需注意）

7. 非功能性需求

• 性能：AI 生成单条视频平均耗时 ≤ 3 分钟（1080p，30秒素材）。队列处理能力 ≥ 10 条/小时（单 GPU）。

• 可靠性：生成失败率 ≤ 5%。付费用户享有 99.9% 的成功率保证（失败全额退款）。

• 存储：视频文件存储在对象存储（OSS），保留至少 1 年。用户可主动删除。

• 安全：视频仅限本家族成员访问（通过 token 鉴权）。分享链接有效期 7 天。

8. 验收标准（DoD）

1. 选择一位有 10 张以上照片的成员，点击生成后，系统成功创建排队任务。
2. 非付费用户看到排队位置和预计时间，付费用户看到立即处理。
3. 生成完成后，视频可在线播放、下载，内容包含照片序列和解说词。
4. 视频中照片按时间顺序排列，直系血脉关系正确。
5. 付费优先功能正常扣款，并跳过排队。
6. 对于无照片的成员，生成前给出明确提示。

9. 与现有模块的关系

• 时光长廊/影像馆：提供素材来源。

• 个人详情页：提供触发入口和目标人物数据。

• 印刷模块：提供二维码印制接口。

• 支付系统：对接微信/支付宝支付（付费优先）。

• AI 服务：新增视频生成适配层，可对接多家供应商。

10. 后续迭代计划

• v1.1：支持用户自定义解说词文案，手动编辑脚本。

• v1.2：支持多人合辑（如“祖孙三代”）。

• v1.4：支持 AI 生成虚拟形象讲述故事（数字人）。
