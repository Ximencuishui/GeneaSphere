# 根脉云谱（GeneaSphere）前端页面设计规范 v1.0

> **版本**：V1.0
> **日期**：2026-06-20
> **适用项目**：GeneaSphere 根脉云谱在线族谱平台
> **技术栈**：Vue 3 + Vite + Element Plus + AntV G6 + Pinia

---

## 一、设计风格定位

根脉云谱作为一款文化传承类产品，设计风格应当体现**庄重、典雅、现代**的气质，融合传统文化内涵与现代交互体验。

### 1.1 整体风格

采用**新中式现代风格**，以素雅色调为主，配合传统纹样元素和现代卡片式布局，营造出既有文化厚重感又不失现代便捷性的用户体验。

## 五、关键页面设计规范

### 5.1 登录/注册页

**布局要求**：
- 全屏背景：使用淡雅的中国风背景图（宣纸纹理、水墨山水）
- 中央登录卡片：宽度 400px，半透明毛玻璃效果
- 卡片边框：1px solid `#C9A96E`（暗金色）

**表单规范**：
- 输入框高度：40px
- 输入框边框：1px solid `#DCDFE6`
- 输入框聚焦：边框变 `#C9A96E`，阴影 `0 0 0 2px rgba(201, 169, 110, 0.2)`
- 按钮高度：40px，背景 `#5D4037`，文字白色

**交互效果**：
- 输入框聚焦时：暗金色高亮动画（0.3s transition）
- 登录按钮悬停：背景变深 `#4A3229`

---

### 5.2 族谱树页面

**顶部工具栏**：
- 高度：56px
- 背景：`#FFFFFF`，底部边框 `1px solid #EBEEF5`
- 工具按钮：图标 + 文字，间距 16px

**画布区域**：
- 背景色：`#FAF8F5`
- 网格（可选）：淡色网格线 `rgba(0,0,0,0.05)`

**树节点样式**（AntV G6 自定义节点）：

```javascript
// 节点尺寸
const NODE_SIZE = {
  width: 120,
  height: 60,
  radius: 8
};

// 节点颜色
const NODE_COLOR = {
  male: '#4A90D9',      // 男性：蓝色
  female: '#E8837C',    // 女性：红色
  selected: '#C9A96E',  // 选中：暗金色
  ancestor: '#5D4037'    // 祖先：深棕
};
```

**右侧详情面板**：
- 宽度：360px
- 内边距：24px
- 头像尺寸：80px × 80px，圆形裁剪

---

### 5.3 导入向导页面

**分步指示器**（使用 Element Plus Steps 组件）：
- 步骤数：3 步
- 当前步骤：高亮 `#C9A96E`
- 已完成步骤：绿色 `#67C23A`

**Step 1 - 上传文件**：
- 拖拽区域：边框 `2px dashed #DCDFE6`，悬停时变 `#C9A96E`
- 支持格式：`.xlsx`, `.xls`
- 最大文件：10MB

**Step 2 - 列映射**：
- 表格布局：左侧 Excel 列名，右侧下拉选择目标字段
- 必填字段标记：红色星号 `*`
- 示例映射：
  - Excel 列 `姓名` → 目标字段 `full_name`
  - Excel 列 `性别` → 目标字段 `gender`
  - Excel 列 `出生日期` → 目标字段 `birth_date`

**Step 3 - 数据预览**：
- 表格高度：400px，超出滚动
- 错误行高亮：背景 `#FEF0F0`
- 操作按钮：`< 上一步`  `下一步 >`  `确认导入`

### 5.6 个人详情"博客页"

## 六、交互动效规范

### 6.1 页面切换

- 过渡效果：`fade`
- 持续时间：300ms
- 缓动函数：`ease-in-out`

```css
.fade-enter-active, .fade-leave-active {
  transition: opacity 0.3s ease;
}
.fade-enter, .fade-leave-to {
  opacity: 0;
}
```

### 6.2 卡片悬停

- 位移：`translateY(-4px)`
- 阴影：`0 4px 16px rgba(0,0,0,0.15)`
- 过渡：all 0.3s ease

### 6.3 树节点点击

- 高亮动画：脉冲效果（pulse）
- 持续：1s，无限循环（直到取消选中）

```javascript
// AntV G6 节点选中动画
node.animate({
  lineWidth: 3,
  stroke: '#C9A96E'
}, {
  duration: 1000,
  repeat: true
});
```

### 6.4 保存成功提示

- 位置：右上角
- 持续时间：3s
- 类型：Element Plus Message 组件

## 七、响应式设计断点

| 设备 | 宽度范围 | 布局调整 |
|------|----------|-----------|
| 桌面端 | > 1200px | 完整布局 |
| 平板端 | 768px ~ 1200px | 左侧导航可收起（折叠为图标） |
| 移动端 | < 768px | 底部 Tab 导航，树形图全屏展示 |

### 移动端适配要点

- 侧边栏改为底部 Tab（56px 高度）
- 族谱树页面改为单栏布局（树形图全屏，详情页滑出）
- 表单元素高度调整为 44px（适应手指点击）

## 十、代码规范

### 10.1 Vue 组件命名

- 组件名使用 **PascalCase**（如 `GenealogyTree.vue`）
- 页面组件加 `Page` 后缀（如 `LoginPage.vue`）
- 通用组件加 `Base` 前缀（如 `BaseButton.vue`）

### 10.2 样式命名

使用 **BEM 命名法**：

```css
/* Block */
.tree-page { }

/* Element */
.tree-page__toolbar { }
.tree-page__canvas { }

/* Modifier */
.tree-page__node--selected { }
```

### 10.3 注释规范

```typescript
/**
 * @description 创建人员并维护闭包表
 * @param data - 人员基本信息
 * @param parentId - 父节点 ID（可选）
 * @returns 创建成功的人员对象
 */
async createPerson(data: CreatePersonDto, parentId?: bigint) {
  // ...
}
```

## 十一、验收标准

### 11.1 视觉验收

- [ ] 所有页面色彩符合色彩体系规范
- [ ] 字体使用正确，无乱码或降级显示
- [ ] 响应式布局在各断点下显示正常
- [ ] 交互动效流畅，无卡顿

### 11.2 交互验收

- [ ] 所有按钮、链接可点击区域 ≥ 44px × 44px（移动端）
- [ ] 表单验证提示清晰，位置合理
- [ ] 加载状态有明确反馈（Loading 或骨架屏）
- [ ] 错误提示友好，提供解决方案

### 11.3 性能验收

- [ ] 首屏加载时间 < 2s（桌面端）
- [ ] 族谱树渲染（1000节点）< 1s
- [ ] 图片懒加载生效，首屏图片优先加载
