# 背景音乐资源目录

## 音频文件说明

本目录存放背景音乐功能所需的音频资源文件。

### 必需文件

1. **night-crickets.mp3** - 夜间虫鸣音效
   - 时长：3-5 分钟
   - 格式：MP3, 128kbps
   - 内容：蟋蟀、纺织娘等昆虫的叫声

2. **field-frogs.mp3** - 田野蛙声音效
   - 时长：3-5 分钟
   - 格式：MP3, 128kbps
   - 内容：稻田青蛙的叫声

3. **summer-night.mp3** - 夏日夜晚混合音效
   - 时长：3-5 分钟
   - 格式：MP3, 128kbps
   - 内容：虫鸣 + 蛙声 + 微风的混合环境音

### 音频资源获取

#### 免费音源推荐

1. **Freesound.org** (CC0 协议)
   - 搜索关键词：`crickets night`, `frog chorus`, `summer evening ambience`
   - 示例链接：
     - https://freesound.org/search/?q=crickets+night&f=license:"Creative+Commons+0"
     - https://freesound.org/search/?q=frog+chorus&f=license:"Creative+Commons+0"

2. **Pixabay** (免费商用)
   - https://pixabay.com/sound-effects/search/crickets/
   - https://pixabay.com/sound-effects/search/frogs/

3. **Zapsplat** (免费，需署名)
   - https://www.zapsplat.com

### 音频处理要求

1. 使用 Audacity 或其他工具进行降噪处理
2. 确保首尾可无缝循环（使用 Crossfade 技术）
3. 导出为 MP3 格式，比特率 128kbps
4. 文件大小控制在 2MB 以内

### 临时占位方案

在获取正式音频资源前，可以：
1. 使用在线音频生成器创建测试音效
2. 从免费音源下载示例文件
3. 使用静音文件测试功能逻辑
